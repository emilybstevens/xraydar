Images of Performance

    When a CXR image is scanned, it is scanned for multiple diseases at once. The resulting output focuses on only true positives, decreasing the risk of accidental misdiagnosis. A graph like the one seen on the left is created for each diagnosis possibility identified for a CXR, delineating the likelihood of the true positive being incorrect.

    Ideally, the upper left quadrant will have the greatest value, indicating that it truly is a true positive. In this case, the image indicates that the patient likely has a positive pneumothorax (collapsed lung), and would be classified as an emergent patient requiring immediate surgery.
